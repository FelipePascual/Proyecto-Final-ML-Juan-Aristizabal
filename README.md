# Proyecto-Final-ML-Juan-Aristizabal
MODELO SUPERVISADO CON EL DATASET "taxis"

- Este repositorio tiene cómo objetivo resguardar y preservar el desarrollo de un modelo supervisado de Random Forest Regressor. Este modelo nace para intentar predecir el costo base de una carrera de taxi en ciertas zonas de EEUU.

- Para su ejecución es recomendado abrir el archivo .ipynb en un Notebook de Google Colab.
